# Proyecto Final equipo PIPE

Análisis de la Encuesta Nacional de Victimización y Percepción sobre Seguridad Pública (ENVIPE) para la República Mexicana y Querétaro.

# Autores del proyecto: Emmanuel Espejel, Carolina Hernández y Julián Martínez

## Archivos

- ProyectoFinal.Rmd: código fuente del reporte.
- ProyectoFinal.pdf: reporte final.
- ProyectoPIPE.Rproj: proyecto de RStudio.
- qro.csv: archivo de mapas

## Instrucciones de ejecución

1. Descargar las bases de datos de la ENVIPE 2021–2025 desde el portal de INEGI: https://www.inegi.org.mx/programas/envipe/ (archivos CSV disponibles en la sección de microdatos).

2. Identificar la ubicación de las carpetas descargadas en el paso anterior.

3. Abrir el archivo `ProyectoFinal.Rmd`.

4. En las líneas 16 a 20, sustituir las rutas de las cinco carpetas de datos por las rutas correspondientes a la ubicación de las bases de datos descargadas.

5. Modificar la ruta especificada en la línea 22 para que apunte a la carpeta `Qro/`, incluida en este repositorio.

6. Ejecutar el documento `ProyectoFinal.Rmd` o generar el archivo PDF mediante la opción **Knit** de RStudio.

## Fuente de datos

Datos obtenidos de la ENVIPE 2021–2025 publicada por INEGI.
